# Pakistani Politician CNN Classifier
![CI/CD](https://github.com/abdul-rehman-s/pakistani-politicians-cnn-classification/actions/workflows/ci.yml/badge.svg)

CNN-based image classification system for Pakistani politicians.

## Technologies
- TensorFlow
- EfficientNetB0
- ResNet50
- DVC
- MLflow
- FastAPI
- Docker
- GitHub Actions

## Dataset
16 classes of Pakistani public figures.
